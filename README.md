# desafio_telecomx_parte2
Projeto de análise e previsão de churn em empresa de telecomunicações, desenvolvido em Python no Google Colab. Utiliza Pandas, NumPy, Matplotlib, Seaborn e Scikit-learn para análise de dados, engenharia de atributos e modelos de ML. Challenge da plataforma Alura, com foco em apoiar estratégias de retenção.
